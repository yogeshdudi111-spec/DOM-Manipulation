# 🎯 DOM Manipulation Mini Project

A simple JavaScript project demonstrating basic **DOM Manipulation** concepts using HTML, CSS, and JavaScript.

This project performs different actions on button clicks, making it a beginner-friendly example for learning JavaScript DOM methods.

---

## 🚀 Features

- 👋 Change displayed user name
- 📊 Toggle attendance percentage
- 🌙 Dark Mode / Light Mode
- ❤️ Like and Unlike functionality
- 🔔 Alert messages for user interactions
- 🎨 Responsive card-based UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
